计算公式验证

1. totalSlippage 计算

const totalSlippage = paperTradingSession.tradesDuringSession.reduce(
  (sum, trade) => sum + Math.abs(trade.slippage || 0),
  0
);

解释：累加所有交易的滑点金额绝对值

2. totalCommission 计算

const totalCommission = paperTradingSession.tradesDuringSession.reduce(
  (sum, trade) => sum + Math.abs(trade.commission || 0),
  0
);

解释：累加所有交易的手续费金额绝对值

3. executionGap 计算

const executionGap = backtestReturn - paperTradingReturn;

解释：backtest回报率减去paper trading回报率

预期输出示例

  • 执行差距：2.34% (backtest比paper trading好2.34%)
  • backtest回报率：5.67%
  • paper trading回报率：3.33%
  • 交易数量：5笔
