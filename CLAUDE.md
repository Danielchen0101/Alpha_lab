# professional_quant_platform

Project skills for Claude/OpenClaw agents:

- **quant-platform-task-runner** — All coding, debugging, and modification tasks
  - `references/frontend-workflow.md` — React/TypeScript rules
  - `references/backend-workflow.md` — Flask rules
  - `references/scanner-ai-rules.md` — Market Scanner, AI, Continue Scan, Quick Backtest rules
  - `references/report-format.md` — Compact report format

- **quant-webapp-validation** — UI verification via browser testing
  - `references/ui-checklist.md` — Scanner/UI validation checklist

## Core rules

1. Read real code first, never guess.
2. Make the smallest scoped change — never rewrite entire pages.
3. Never delete working features.
4. Never replace real logic with mock/static placeholders.
5. Run `npm run build` after every frontend change.
6. Always report compactly with changed snippets.
